The Infinity Blade Save Dumper tool aims to turn Serialized Infinity Blade Save Packages into a readable Json format so the user can make highly customizable changes. The Dumper also has a feature that re-serializes your changed save data, making it fast and easy tochaange and import into your game.

Im aiming to have support for all 4 ChAIR games by the time this is finished, and am currently in the midst of completely rewriting my codebase for a faster and cleaner experience.
